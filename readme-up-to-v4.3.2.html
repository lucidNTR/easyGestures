Overview

It is a pie menu that pops up inside the browser when you click the mouse and then let you perform various actions
at the tip of the mouse pointer with minimum mouse movement.

The fast straight mouse gestures you perform this way save you from reaching toolbars, menus and keyboard and
keep you from having to memorize any gestures. All behaviors and actions are customizable.

This is an extension for Mozilla Firefox Browser.

Localized in English(en-US) and French(fr-FR) and several other languages.

System requirements

To run easyGestures, your system must have Mozilla Firefox 3.6 installed.

Uninstallation

To uninstall easyGestures, use Firefox Extension Manager.

Display of the pie menu:
* is limited to the current frame.
* doesn't work on top of plugins (ex: Flash).
